# 🚀 Shelf-Win: Your Windows Desktop, Organized.

**Shelf-Win**, Windows masaüstünüzü karmaşadan kurtaran, ekranın kenarında yaşayan, şık ve fonksiyonel bir "floating dock" (yüzen raf) uygulamasıdır. Metin parçacıklarını, dosyaları ve uygulama kısayollarını sadece sürükleyip bırakarak organize etmenizi sağlar.

[English](#english) | [Türkçe](#türkçe)

---

## 🇹🇷 Türkçe

### ✨ Özellikler
* **Sürükle-Bırak:** Metinleri seçip rafa bırakın, anında `.txt` notu olarak kaydedilsin.
* **Uygulama Kısayolları:** `.exe` veya dosyaları sürükleyin, ikonları otomatik çekilsin.
* **Modern Arayüz:** Windows 11 Acrylic/Mica efektli, minimalist tasarım.
* **Her Zaman Üstte:** Pencerelerin arasında kaybolmaz, hep elinizin altındadır.
* **Çoklu Raf:** Farklı işleriniz için birden fazla dock oluşturun.

### 🛠️ Kurulum
1. [Releases](https://github.com/TuncerUgur/Shelf-Win/releases) sayfasından en güncel `.zip` dosyasını indirin.
2. Klasöre çıkartın ve `DockShelf.exe` dosyasını çalıştırın.


## 🏗️ Teknik Özellikler
* **Dil:** C#
* **Framework:** .NET 8 / WPF
* **UI Style:** WinUI-inspired, Borderless, Transparent
* **Storage:** Local JSON settings & Local File System

## 🇺🇸 English

### ✨ Features
* **Drag & Drop:** Select any text and drop it on the shelf to save it as a `.txt` snippet instantly.
* **App Shortcuts:** Drop `.exe` files or documents; icons are fetched automatically.
* **Modern UI:** Minimalist design with Windows 11 Acrylic/Mica transparency effects.
* **Always on Top:** Stays accessible on the side of your screen at all times.
* **Multi-Dock Support:** Create multiple shelves for different categories (Work, Dev, Social).

### 🛠️ Installation
1. Download the latest `.zip` file from the [Releases](https://github.com/TuncerUgur/Shelf-Win/releases) section.
2. Extract the folder and run `DockShelf.exe`.
